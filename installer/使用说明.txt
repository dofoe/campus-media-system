========================================
校园宣传素材智能管理系统
一键安装版 - 使用说明
========================================

【一、系统要求】

操作系统: Windows 10 / Windows 11 / Windows Server 2016+
处理器:   x64 架构
内存:     最低 2GB，推荐 4GB
硬盘:     至少 500MB 可用空间
浏览器:   Chrome 90+ / Edge 90+ / Firefox 88+

【二、安装步骤】

方式一：一键安装（推荐）
1. 双击 "CampusMediaSystemSetup-1.0.0.exe"
2. 按照安装向导完成安装
3. 安装完成后自动启动系统

方式二：绿色版（免安装）
1. 解压本压缩包到任意目录
2. 双击 "start.bat" 启动系统
3. 浏览器访问 http://localhost:8080

【三、首次使用】

1. 打开浏览器访问 http://localhost:8080
2. 使用默认账号登录：
   用户名: admin
   密码:   admin
3. 登录后请立即修改默认密码
   路径: 管理后台 → 用户管理 → 编辑 → 重置密码

【四、注册为Windows服务】

如果通过绿色版运行，可以注册为Windows服务：
1. 右键 "install-service.bat"
2. 选择 "以管理员身份运行"
3. 服务将自动启动并设为开机自启

【五、卸载系统】

方式一：通过控制面板
- 控制面板 → 程序和功能 → 卸载程序
- 找到"校园宣传素材智能管理系统" → 卸载

方式二：通过安装包
- 双击运行安装包
- 选择"卸载"选项

方式三：手动卸载
1. 右键 "uninstall-service.bat" → 以管理员身份运行
2. 删除整个安装目录

【六、数据备份与恢复】

备份：
- 复制 data 目录（数据库）
- 复制 uploads 目录（上传的文件）

恢复：
- 将备份的 data 和 uploads 目录覆盖到安装目录
- 重启服务

【七、技术支持】

- 项目地址: https://github.com/dofoe/campus-media-system
- 问题反馈: GitHub Issues
